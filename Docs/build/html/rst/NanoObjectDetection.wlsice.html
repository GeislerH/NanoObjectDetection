NanoObjectDetection.wlsice module¶

NanoObjectDetection.wlsice.chi2(params, t, y, R)¶

Compute chi^2 function to be minimized, based on input (y-f)*R*(y-f) This gives us our fitted parameters in params. (If R=diag(inv(Cov)) we get least square fit)

NanoObjectDetection.wlsice.chi2Jacobian(params, t, y, R)¶

Jacobian of chi^2 function, i.e. -df/dparams[i]*[C^(-1)]*(y-f) + (y-f)*[C^(-1)]*(-df/dparams[i]) return np.array of result for each i
